# 🧠 00_memory — 记忆模块

> AI Agent 的核心记忆系统，为所有模块提供上下文记忆支持。

---

## 模块概述

记忆模块是 AIforXuX 系统的"大脑存储区"，负责管理 AI 在工作过程中产生和需要的各类记忆信息。

## 记忆分类

### 📦 long_term — 长期记忆
- **用途**：存储核心知识、技能经验、长期有效的信息
- **特点**：永久保留，手动管理
- **示例**：用户偏好、项目核心规则、经验教训

### ⭐ important — 重要记忆
- **用途**：存储关键决策、重大事件、高优先级信息
- **特点**：永久保留，标记优先级
- **示例**：重要会议决定、关键项目节点、紧急事项

### 🔄 auto — 自动记忆
- **用途**：对话和操作过程中自动沉淀的信息
- **特点**：定期清理，AI 自动判断是否保留
- **示例**：对话中提到的临时信息、操作记录

### 💨 short_term — 短期记忆
- **用途**：当前会话级别的上下文信息
- **特点**：会话结束后清理
- **示例**：当前对话上下文、临时任务状态

## 记忆条目格式

每条记忆存储在 `entries/` 目录下，建议使用以下格式：

```markdown
# 记忆条目标题

- **创建时间**：YYYY-MM-DD HH:MM
- **来源模块**：模块编号
- **优先级**：高/中/低
- **标签**：#标签1 #标签2

## 内容
（记忆正文）

## 关联
（相关记忆/模块的链接）
```
