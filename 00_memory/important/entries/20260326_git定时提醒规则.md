
# 定时提醒 — Git 同步提醒规则

- **创建时间**：2026-03-26 11:08
- **来源模块**：全局
- **优先级**：高
- **标签**：#Git #定时提醒 #每日必做

## 内容

### ⏰ 早上提醒（10:00 上班后，首次打开工具时）

> **🔔 提醒：你好！新的一天开始了，请先执行 `gitpull` 拉取最新内容，再开始工作！**

```bash
git pull origin main
```

- **触发条件**：早上 10:00 之后，用户首次开启 AI 工具 / 开始会话时
- **提醒内容**：先同步 GitHub 上的最新内容，避免本地版本落后

---

### ⏰ 下午提醒（18:00 之后）

> **🔔 提醒：快下班了！请记得执行 `gitpush` 上传今天的工作成果，确保远程备份！**

```bash
git add -A && git commit -m "update: 日常更新 $(date)" && git push
```

- **触发条件**：下午 18:00 之后，用户与 AI 工具交互时
- **提醒内容**：将今天所有变更提交并推送到 GitHub，防止数据丢失

---

## AI 执行规则

> **任何 AI 在以下情况下，必须主动触发提醒：**
>
> 1. **当前时间 ≥ 10:00 且为今日首次会话** → 提醒执行 `gitpull`
> 2. **当前时间 ≥ 18:00** → 提醒执行 `gitpush`
> 3. **每次与 AI 协作结束后** → 提醒执行 `gitpush` 保存会话产出
