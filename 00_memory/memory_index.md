## 📝 记忆条目格式规范

新建记忆条目请使用以下格式，文件命名：`YYYYMMDD_简短标题.md`

```markdown
# 记忆条目标题

- **创建时间**：YYYY-MM-DD HH:MM
- **来源模块**：模块编号（如 01_game_info）
- **优先级**：高 / 中 / 低
- **标签**：#标签1 #标签2

## 内容
（记忆正文，简洁准确）

## 关联
（相关记忆文件或模块链接）
```

- **新增记忆后**：立即在本文件对应表格中添加索引条目
