# 🎮 01_game_info — 游戏信息资料收集整理

> 负责游戏相关信息、资料、内容的收集、整理和管理。

---

## 模块概述

本模块专注于游戏行业信息的全方位收集与整理，包括但不限于：竞品分析、市场动态、玩法机制研究、美术风格参考等。

## 子目录说明

| 目录 | 说明 |
|------|------|
| `collectors/` | 信息采集器 — 爬虫脚本、API 对接、数据抓取工具 |
| `processors/` | 数据清洗与整理 — 格式转换、去重、分类处理 |
| `storage/` | 数据存储 — 本地文件/数据库存储方案 |
| `resources/` | 参考资料 — 素材库、参考文档、截图等 |

## 工作流程

```
信息源 → collectors（采集）→ processors（清洗整理）→ storage（存储）
                                                        ↓
                                                   resources（归档）
```
