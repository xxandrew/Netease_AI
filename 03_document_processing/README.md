# 📄 03_document_processing — 文档处理

> 负责 Office 文档（Word/Excel/PDF/PPT）的自动化处理、模板管理，以及 AI 辅助文档转换等。

---

## 模块概述

本模块提供 AI 辅助的文档生产能力，覆盖 PPT 制作、图片文字提取、模板管理等场景，旨在大幅缩短文档制作周期。

## 🔄 PPT 制作快速入口

想做 PPT？参考 [PPT制作工作流.md](ppt/PPT制作工作流.md)，用以下格式启动：

```
我要做一个PPT：
主题：[一句话说清楚]
目标：[这个PPT要达到什么效果]
受众：[谁来看]
时间：[汇报时长]
```

AI 会接管大纲 → 内容 → 讲稿的完整流程。
