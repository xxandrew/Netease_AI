## 🛠️ 制作PPT的三种方式

### 方式一：AI 生成内容 + 手动填入PPT（推荐日常使用）
- 用 AI 按模板生成每页文字内容
- 手动复制到 PowerPoint/WPS 对应页面
- **适合**：有固定公司模板，只需填内容的场景

### 方式二：AI 生成 Markdown → 转PPT工具导入
- AI 输出结构化 Markdown
- 用工具一键转为PPT：
  - **Gamma.app**（在线，效果最好）：https://gamma.app
  - **MindShow**（国内，免费）：https://www.mindshow.fun
  - **Beautiful.ai**：https://www.beautiful.ai
- **适合**：需要快速生成一份新PPT，不在乎模板风格

### 方式三：直接用 AI PPT工具生成（全自动）
| 工具 | 特点 | 地址 |
|------|------|------|
| **Gamma** | 效果最好，支持中文 | gamma.app |
| **MindShow** | 国内访问稳定，免费 | mindshow.fun |
| **AiPPT** | 国内，专为PPT设计 | aippt.cn |
| **讯飞智文** | 中文最优，科大讯飞出品 | 1.cn/zhiwen |

---

## 📌 使用建议

> 1. **优先用方式一**：保持公司PPT风格统一，AI 只负责内容
> 2. **模板是核心**：每次使用前先打开对应模板，按框架填信息
> 3. **多迭代**：AI 第一版不满意，直接说「调整第X页，改为...」
> 4. **保存提示词**：好用的提示词记录到 `prompts/PPT_AI提示词库.md`
