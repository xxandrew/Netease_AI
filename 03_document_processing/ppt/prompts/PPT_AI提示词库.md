3. 把【填入】的部分替换成你的实际内容
4. 发送给 AI，获取结果

## ✏️ 通用类提示词

### P-001｜按模板生成PPT内容
```
我有一个PPT需要制作，请按照以下结构帮我生成每页的完整内容。

PPT主题：【填入】
目标受众：【填入，如：部门领导/项目团队/外部客户】
汇报时长：【填入，如：10分钟/20分钟】

PPT结构（逐页说明）：
第1页：封面 - 【填入标题和副标题】
第2页：【填入页面主题和要点】
第3页：【填入页面主题和要点】
...

要求：
- 每页内容控制在3-5个要点
- 语言简洁专业，避免口语化
- 标注每页建议配什么图表
- 最后给出整体讲稿备注
```

---

### P-002｜优化已有PPT内容
```
请帮我优化以下PPT内容，让它更清晰、更有说服力。

当前PPT内容：
【把当前PPT的文字内容粘贴在这里】

优化要求：
- 目标受众：【填入】
- 优化重点：【填入，如：让结论更突出/减少文字/加强数据支撑】
- 保持原有结构，只优化文字表达

请逐页输出优化后的版本，并说明优化了哪些地方。
```

### P-003｜将文字内容转为PPT大纲
```
请将以下文字内容整理成PPT大纲结构。

原始内容：
【把你的文字/报告/文档内容粘贴在这里】

要求：
- 提炼核心观点，不超过【填入，如：10】页
- 每页只保留最关键的3-5个要点
- 给出每页的标题和内容框架
- 标注建议配什么类型的图表（表格/图表/图片等）
```

【项目概述】

预计研发周期：【填入】
预计团队规模：【填入】

【申请内容】
本次汇报目的：【填入，如：申请正式立项/申请预研资源/阶段性汇报】
需要的支持：【填入，如：预算XX万/增加XX人/技术支持】

## 📝 会议纪要类提示词

### P-007｜会议笔记整理成纪要
```
请帮我将以下会议记录整理成规范的会议纪要。

【会议信息】
会议主题：【填入】
时间：【填入】
参会人：【填入】

【原始记录】
（把你的笔记/录音文字直接粘贴在这里，格式不整也没关系）：

【整理要求】
请提炼并输出：
1. 会议背景（1-2句）
2. 讨论的主要议题
3. 核心决策事项（每条要清晰，无歧义）
4. 待办事项（含负责人和截止时间，没提到的填"待定"）
5. 遗留问题
6. 下次会议建议议题

输出格式：适合PPT展示的结构化内容，每页内容清晰。
```

## 💡 使用技巧

> **提示词用得好，PPT质量翻倍的关键：**
>
> 1. **信息给得越具体，结果越好** — 别只写「做个PPT」，把背景、受众、目的都说清楚
> 2. **分步骤迭代** — 先让AI生成大纲，确认结构后再让它填内容
> 3. **单页优化** — 哪页不满意就单独发给AI优化，不用重新来过
> 4. **保存好用的变体** — 遇到特别好用的提示词，记录在这个文件里
