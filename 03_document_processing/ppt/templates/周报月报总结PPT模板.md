# 📅 周报 / 月报总结 PPT 模板

> **适用场景**：每周工作汇报、月度/季度总结、部门汇报
> **建议页数**：6 - 10 页
> **汇报时长**：5 - 15 分钟

## 📋 PPT 完整结构模板

---

### 第1页｜封面
```
【周报 / 月报】
汇报人：XXX   部门：XXX
汇报周期：XXXX年XX月XX日 - XXXX年XX月XX日
```

---

### 第2页｜本期工作概览（仪表盘页）
```
标题：本期工作一览

内容框架（推荐用卡片/色块布局）：

📋 计划任务数：X项
✅ 完成任务数：X项（完成率 XX%）
🔄 进行中：X项
⚠️ 延期/风险：X项

关键成果一句话：[本期最重要的1个成果]
```

---

### 第3页｜重点工作完成情况
```
标题：本期重点工作

内容框架（表格或列表）：

| 工作项 | 目标 | 完成情况 | 状态 |
|--------|------|---------|------|
| 工作1 | [目标描述] | [完成情况] | ✅完成 |
| 工作2 | [目标描述] | [完成情况] | 🔄进行中 |
| 工作3 | [目标描述] | [完成情况] | ⚠️延期 |

重点说明：[补充说明1-2个重要工作的细节]
```

---

### 第4页｜关键数据（如适用）
```
标题：本期关键数据

内容框架：
▸ 核心指标1：[名称] [数值] vs 上期 [↑XX% / ↓XX%]
▸ 核心指标2：[名称] [数值] vs 目标 [完成率XX%]
▸ 核心指标3：[名称] [数值] [说明]

趋势分析：[1-2句对数据变化的解读]

建议配图：折线图/柱状图展示趋势
```

---

```

|--------|------|------------|--------|--------|

```

---

### 第7页｜总结（月报/季报适用）
```
标题：本期总结

内容框架：
▸ 最大亮点：[1-2个值得肯定的成果]
▸ 主要挑战：[1-2个主要困难]
▸ 经验沉淀：[可复用的方法论/经验]
▸ 下阶段重点：[下个周期的核心目标]
```

## 💡 制作技巧

| 技巧 | 说明 |
|------|------|
| **数字化表达** | 工作成果尽量用数字量化，如「完成3个版本迭代」而非「做了很多」 |
| **问题与对策成对出现** | 提问题必须配解决方案，不要只抱怨 |
| **重点突出** | 用颜色/图标区分完成/进行中/延期状态 |
| **控制篇幅** | 周报6-8页够了，月报不超过10页 |
| **提前量** | 计划页要比上期完成情况多写1-2项，体现主动性 |
