# 🖼️ 图片文字提取 → MD / Word / Excel

> 利用 AI 视觉能力，将图片中的文字、表格、结构化内容一键转换为可编辑文档，彻底告别手动复制。

---

## 🎯 这个功能能解决什么问题？

在日常办公中，经常遇到这些场景：

| 场景 | 过去的做法 | 用 AI 提取后 |
|------|-----------|------------|
| 收到截图形式的表格/评分表 | 手动一格一格录入 | 30 秒自动提取，直接可编辑 |
| 扫描件/拍照文档需要整理 | 对着图片逐字打字 | AI 识别文字，还原结构 |
| PPT/设计稿中的文字需要归档 | 截图存档，无法搜索 | 转为 MD，可全文搜索 |
| 会议白板照片需要整理成文档 | 手写转录，容易遗漏 | AI 提取文字，自动分段 |
| 竞品 App 截图需要分析记录 | 看图描述，效率低 | 直接提取界面文字归档 |

**核心价值：把「不可编辑的图片」变成「可编辑、可搜索、可复用」的结构化文档。**

---

## ⚡ 快速操作（3步完成）

```
第1步：截图或准备好图片
第2步：发给 AI，说「提取成 MD 文档」
第3步：AI 输出内容，保存为 .md 文件
```

**常用指令模板：**

```
# 单张图片
把这张图片的内容提取成 MD 文档

# 多张图片（指定路径）
C:\Users\用户名\Desktop\pic
把这个目录下的图片逐一提取成 MD 文件，保存在同目录

# 提取后直接保存到项目
把图片内容提取成 MD，保存到 c:\AIforXuX\对应模块目录
```

## 📊 支持的内容类型

| 内容类型 | 效果 | 备注 |
|---------|------|------|
| 纯文字段落 | ⭐⭐⭐⭐⭐ | 完整还原，含段落结构 |
| 表格 | ⭐⭐⭐⭐⭐ | 自动转为 Markdown 标准表格 |
| 标题层级 | ⭐⭐⭐⭐⭐ | 识别字号/加粗，转为 H1-H3 |
| 列表/编号 | ⭐⭐⭐⭐ | 还原有序/无序列表 |
| 混合图文 | ⭐⭐⭐⭐ | 提取文字部分，图片跳过 |
| 手写内容 | ⭐⭐⭐ | 字迹清晰可识别，潦草需校对 |

> 💡 **最佳实践**：使用截图工具（如 Snipaste、微信截图）而非手机拍照，识别率更高。

## 📄 MD 转 Word 方法

### 方法一：Pandoc 命令行（最完整，推荐）

**安装 Pandoc：** https://pandoc.org/installing.html

```bash
# 基础转换
pandoc 文件名.md -o 文件名.docx

# 示例
pandoc 评分标准_5分制.md -o 评分标准_5分制.docx

# 应用自定义 Word 模板（保持公司样式）
pandoc 文件名.md --reference-doc=模板.docx -o 输出.docx
```

**效果**：标题、表格、列表、粗体全部自动转换，表格带边框，开箱即用。

---

### 方法二：直接让 AI 转换（最简单）

```
把这个 MD 文件的内容帮我整理成 Word 格式的结构
（让 AI 重新输出为适合复制到 Word 的排版）
```

然后复制输出内容 → 粘贴到 Word → 应用样式。

---

### 方法三：Typora / Obsidian 导出

1. 用 **Typora** 打开 `.md` 文件
2. 菜单：`文件` → `导出` → `Word (.docx)`
3. 直接生成 `.docx` 文件

> Typora 下载：https://typora.io/

---

### 方法四：VS Code 插件

1. 安装插件：`Markdown PDF` 或 `Office Viewer`
2. 右键 `.md` 文件 → `Export to docx`

---

## 📊 MD 转 Excel 方法

### 方法一：让 AI 直接输出 CSV（最快）

```
把这个 MD 文件中的表格转换成 CSV 格式输出
```

AI 输出 CSV 内容后：
1. 新建一个 `.csv` 文件，粘贴内容保存
2. 用 Excel 打开即可（自动识别列分隔）

---

### 方法二：Pandoc 转 CSV

```bash
# 提取 MD 中的表格并转为 CSV
pandoc 文件名.md -t csv -o 表格.csv
```

---

### 方法三：手动粘贴（适合简单表格）

1. 用 VS Code 或 Typora 打开 `.md` 文件
2. 复制表格内容
3. 打开 Excel → 粘贴到单元格
4. 选择「使用文本导入向导」→ 以 `|` 为分隔符

---

### 方法四：在线工具转换

| 工具 | 地址 | 特点 |
|------|------|------|
| Tables Generator | tablesgenerator.com | MD 表格 → Excel/CSV，在线免费 |
| CloudConvert | cloudconvert.com | 支持 MD → XLSX 直接转换 |
| Markdown to Excel | 浏览器搜索即可 | 多个免费在线工具可用 |

## ⚠️ 注意事项

- 提取后建议**快速人工校对**，尤其是数字和专有名词
- 颜色、字体等视觉样式不会保留在 MD 中，仅保留文字结构
- 含敏感信息的图片建议使用本地 AI 工具（如本地部署的 Ollama）处理
- Word/Excel 转换后，复杂排版可能需要手动微调
