# 📚 05_knowledge_base — 知识库管理

> 负责企业/项目知识库的构建、索引和 RAG 检索问答。

**状态**：🟡 规划中 — 基础结构已建，RAG 知识库待展开。

---

## 模块概述

本模块专注于将分散的项目知识（文档、经验、规范、决策）结构化沉淀，并通过 RAG（检索增强生成）技术实现智能检索问答，让 AI 能直接回答"我们之前怎么做的"类问题。

---

## 规划功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 📂 **知识入库** | 将文档、笔记、规范自动解析入向量数据库 | 🔴 高 |
| 🔍 **智能检索** | 基于语义搜索，快速定位相关知识条目 | 🔴 高 |
| 💬 **知识问答（RAG）** | 基于本地知识库回答项目相关问题 | 🔴 高 |
| 🏷️ **分类标签系统** | 多维度标签体系（项目/模块/时间/类型）| 🟡 中 |
| 📊 **知识图谱** | 可视化知识关联关系，发现隐性连接 | 🟢 低 |

---
